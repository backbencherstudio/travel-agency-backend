# travel-agency-backend

Travel agency backend system created using nestjs

## Installation

Install all dependencies

```
yarn install
```

Seed dummy data to database

```
yarn cmd seed
```

## Running:

Running in development:

```
yarn start:dev
```

Running in production:

First build the project.

```
yarn build
```

Then run.

```
yarn start:prod
```

- Node.js
